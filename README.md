## 🚀 Getting Started

### Prerequisites
No special prerequisites required. The portfolio runs directly in any modern web browser.

### Installation
1. Clone or download the project files
2. Open `index.html` in your preferred web browser
3. The portfolio is ready to use!

### Customization
To customize the portfolio for your own use:

1. **Update Personal Information**
   - Edit the hero section with your name and details
   - Update the about section with your professional summary
   - Modify contact information

2. **Customize Skills**
   - Update skill names and percentages in the skills section
   - Modify progress bar values in HTML

3. **Add Projects**
   - Update project cards with your own projects
   - Replace placeholder images and descriptions
   - Update technology tags

4. **Brand Colors**
   - Modify CSS variables in `style.css` for color scheme
   - Update accent colors and gradients
